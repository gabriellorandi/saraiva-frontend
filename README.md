## SARAIVA EDUCACAO

Interface de busca receitas de coquetéis e drinks.

## Tecnológias utilizadas

- [Reactjs](https://reactjs.org/). 

## Instalaçao

### Linux (Ubuntu)
 
 ```shell
 sudo apt update
 sudo apt install npm
 npm install react react-dom --save
 npm install
 npm start
 ```
